Financial Data

The purpose of this website is to collect interesting financial data all over the world which can help you do investment on different kinds of assets (e.g. stock, bond etc.) We are particularly interested in collecting the data which can be used to evaluate and compare the relative valuation of those assets, such as pe ratio etc.

Currently, we only collect data on some iShare ETFs that target markets of several major countries. More data will be available in the future.
